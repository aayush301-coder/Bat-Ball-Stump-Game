# 🏏 Bat Ball Stump Game

A fun interactive browser game inspired by the classic hand game concept — built using **HTML, CSS, and JavaScript** with persistent score tracking and a modern UI.

---

## 🚀 Live Gameplay Concept

Choose one of the three options:

* **Bat**
* **Ball**
* **Stump**

The computer randomly selects its move, and the winner is decided based on predefined rules.

---

## 🎯 Game Rules

| Player | Beats |
| ------ | ----- |
| Bat    | Ball  |
| Ball   | Stump |
| Stump  | Bat   |

If both choices match → **Tie**

---

## 🛠️ Tech Stack

* HTML5 — structure
* CSS3 — styling + layout
* JavaScript (Vanilla) — game logic

## 💾 Score System

Scores are saved using **LocalStorage**, meaning:

* refresh page → score stays
* close browser → score stays
* reset button → clears score
